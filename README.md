# tictactoe3

Simple as hell, standalone Tic-Tac-Toe game

Copyright &copy; 2024 Dmitriy Naumov naumov1024@gmail.com

## Prerequisites

- OS: Windows / Linux / MacOS
- node: >= 18
- npm: >= 8

## Usage

### Dev server demo

```
cd front
npm i
npm run dev
```

Open [http://localhost:5000/](http://localhost:5000) in browser and have fun!

### Build

```
cd front
npm i
npm run build
```

Build output will be in front/dist.
